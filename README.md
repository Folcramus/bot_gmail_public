БОТ ДЛЯ ОБРАБОТКИ EMAIL из GMAIL

Бот предназначен для парсинга определенных сообщений из gmail и пересылка форматированных сообщений по топикам в супергруппе ТГ.

Этот бот создан в 2021 году так что некоторые библиотеки могут быть устаревшими.

ВНИМАНИЕ!!! Для корректного запуска вам необходим файлы .env c конфигами бота, Gmail, супергруппы и топиками и 2 json файла с ключами и токенами от акка Google и Gmail. 
В целях безопасности я эти файлы выдам лиду который получит этот проект для тестирования. в .env надо  будет в TELEGRAM_BOT_TOKEN TELEGRAM_GROUP_ID вставить токен бота
и id супергруппы (id начинается с -100) и LABEL_TO_THREAD_MAPPING в котором надо будет проставить id топиков в виде LABEL_TO_THREAD_MAPPING={"income": id, "expense": id, "transfer": id}


бот запускается через файл bot.py, если хотите можете запустить docker контейнер, Dockerfile в проекте есть
